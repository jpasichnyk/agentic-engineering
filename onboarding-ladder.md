## 1. The four levels

- **L1 — chat user.** Questions, explanations, snippet help.
- **L2 — IDE autocomplete.** In-line completion, single-file edits.
- **L3 — single-agent agentic.** Multi-step execution, plans, terminal access. Start with [Permissions and Allowlists](field-guide.md#permissions-and-allowlists) before giving an agent terminal access.
- **L4 — multi-agent / orchestrated.** Clean-context separation, [cross-model review](field-guide.md#cross-model-code-review), conductor-style automation. See [Plan-Then-Execute](field-guide.md#plan-then-execute) and [Choosing Model, Vendor, Effort](field-guide.md#choosing-model-vendor-effort).
